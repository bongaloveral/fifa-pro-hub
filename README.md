# FIFA Pro Hub

一个用于归档和发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于存放与 FIFA 游戏相关的独立 HTML 页面资源。每个页面均为独立的静态文件，不依赖特定后端服务，可直接在浏览器中打开使用。页面内容涵盖数据展示、工具辅助、信息查阅等多种用途，旨在提供轻量、便捷的访问方式。

## 目录结构

```
fifa-pro-hub/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 公共资源文件（如样式、脚本、图片）
├── docs/           # 项目文档或说明文件
└── README.md       # 本文件
```

- `pages/`：每个子目录或单独文件对应一个独立页面，页面之间互不依赖。
- `assets/`：包含多个页面可能共用的 CSS、JavaScript 或图像资源。
- `docs/`：用于存放与项目相关的附加说明、更新日志等。

## 页面归档说明

本仓库中的 HTML 页面均为静态归档，内容基于游戏相关数据整理。每个页面均遵循以下原则：

- **独立性**：单个页面可独立运行，无需额外配置。
- **稳定性**：页面内容不依赖外部 API 或动态数据源，仅使用内嵌数据或本地资源。
- **可访问性**：页面设计尽量简洁，确保在主流浏览器中正常显示。

若需新增页面，请将 HTML 文件放置于 `pages/` 目录下，并确保引用资源路径正确。建议为每个页面添加简短的说明注释。
